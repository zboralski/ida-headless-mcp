## Il2CppDumper IDA Scripts

These helpers are copied verbatim from [Perfare/Il2CppDumper](https://github.com/Perfare/Il2CppDumper) (MIT license). They let the worker import `script.json` metadata and `il2cpp.h` structs directly into an IDA session.

Files included:

- `ida_with_struct.py` – Python 2/IDA 7.x entrypoint
- `ida_with_struct_py3.py` – Python 3/IDA 8.x entrypoint
- `LICENSE` – Original Il2CppDumper MIT license

When bundling these scripts in builds or distributing binaries, keep the LICENSE alongside them per MIT requirements.
